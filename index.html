Network Planning Tools

Single-file HTML tools for 2G frequency planning and 4G PCI / RSI / TAC planning. No installation required — open in any modern browser.

2G · v3 ⭐

2G Frequency Planner v3

The v2 engine plus optional BCF ID / BTS ID assignment: free decade blocks on the 10x+1 grid, checked against every BCF and BTS ID in the site's BSC.

  • BCF proposed only when its whole decade N…N+9 is free in the BSC
  • Dual-band BTS numbering: 900 from N, 1800 from N+4
  • Optional BCF/BTS columns auto-detected from CMExport
  • Best-server footprint T1/T2 neighbour engine (from v2)
  • C/I-based BCCH selection (from v2)
  • BCF/BTS IDs included in the XLSX export

2G · v2

2G Frequency Planner v2

Next-generation 2G planner: neighbours from best-server dominance footprints instead of angular heuristics, C/I-based BCCH selection, and a self-repairing multi-pass flow.

  • Best-server footprint T1/T2 — visible & verifiable on the map
  • Dual-band co-located cells handled as one radiator
  • BCCH chosen by worst-decile C/I over the sector's own footprint
  • Hardest-first ordering, Gauss-Seidel passes, targeted repair pass
  • BSIC slot with maximum true repeat distance
  • Config & column mappings remembered between sessions

2G

2G Frequency Planner

Assigns BCCH, BSIC (NCC/BCC) and TCH frequencies for new GSM sites against an existing network, respecting reuse distances and BSIC uniqueness constraints.

  • BCCH tier cascade: clean → T2 → T1 → forced
  • BSIC uniqueness enforced without sacrificing BCCH tier
  • TCH six-pass cascade with sector-order shuffling
  • Interactive map with sector markers
  • Progress bar & async planning (UI stays responsive)
  • Export to XLSX

4G

4G PCI / RSI / TAC Planner

Assigns LTE Physical Cell ID (PCI), Root Sequence Index (RSI) and Tracking Area Code (TAC) for new eNB sites, with MOD3 group enforcement and RSI gap constraints.

  • PCI MOD3 rule: unique mod3 class within each block of 3 sectors
  • Radius reduction to satisfy MOD3 (never below MinReuse)
  • RSI intra-site uniqueness + configurable minimum gap
  • TAC inherited from nearest existing site
  • Progress bar & async planning (UI stays responsive)
  • Export to XLSX with QA columns
